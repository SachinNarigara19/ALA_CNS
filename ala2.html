What is Hashing?

A hash function takes an input of any length and returns a fixed-length string of characters. It is a "one-way" mathematical function.

Integrity & signatures

Hashing ensures data integrity. In digital signatures, we hash the message first because encrypting large data is slow. Changing the data changes the hash, breaking the signature.
